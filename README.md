## 🧠 Sobre o Projeto

Este repositório contém minha jornada na **Piscine Python for Data Science - 0 - Starting**.

Mais do que aprender Python, este projeto representa:

* sobreviver ao peer-evaluation;
* escrever código limpo;
* evoluir de scripts simples para um pacote Python instalável.

**Você reprograma seu cérebro.**

---

## ⚙️ Regras do Projeto

Seguindo as leis sagradas da Piscine:

* ✅ Python **3.10**
* ✅ Sem variáveis globais
* ✅ Imports explícitos (`import numpy as np`)
* ✅ Todo programa possui `main()`
* ✅ Todas as funções possuem `docstring`
* ✅ Nenhum crash inesperado permitido
* ✅ Código validado com **flake8**

Instalar o norm:

```bash
pip install flake8
flake8 .
```
> **Se passa no teste mas falha no norm → ainda está errado.**

## 🧩 Exercícios

---

### ✅ ex00 — Primeiro Script Python

Manipulação básica de:

* listas
* tuplas
* sets
* dicionários

Primeiro contato com a filosofia:

```
Python também sofre mutação.
```

---

### ⏱️ ex01 — Formatação de Tempo

Aprendizado sobre:

* timestamps
* data atual
* notação científica

---

### 🔎 ex02 — Tipos de Objetos

Tudo em Python é objeto.

Até seus erros.

Conceitos:

* `type()`
* funções
* retorno de valores

---

### 🕳️ ex03 — NULL Not Found

Explorando os diferentes tipos de "nada":

* `None`
* `NaN`
* string vazia
* zero
* booleano

Aprendendo que **nem todo vazio é igual**.

### 🔢 ex04 — Par ou Ímpar

Primeiro programa com argumentos via terminal.

Conceitos:

* `sys.argv`
* validação

### 🏗️ ex05 — Analisador de Texto

Programa standalone que analisa:

* letras maiúsculas

* espaços

### 🧬 ex06 — Recriando o `filter`

### 📡 ex07 — Codificador Morse

Reimplementação da barra de progresso `tqdm`.

* build
